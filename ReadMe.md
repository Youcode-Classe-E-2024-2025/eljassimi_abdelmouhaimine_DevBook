# 📚 DevBook — Gestionnaire de livres techniques pour développeurs

DevBook est une application web conçue pour aider les développeurs à organiser et suivre leurs lectures techniques : livres de programmation, design, architecture logicielle, et bien plus encore.

Ce projet est une opportunité pédagogique de concevoir une application **full stack** sans framework, en utilisant uniquement les bases : JavaScript, SQL, Programmation Orientée Objet et UML.

---

## 🚀 Fonctionnalités principales

- ✅ Ajouter, modifier et supprimer des livres.
- 📂 Organiser les livres par catégories (JavaScript, UI/UX, Backend, etc.).
- 📖 Suivre les statuts de lecture : lu / en cours / à lire.
- 🔍 Rechercher et trier la liste de livres.
- 📊 Pagination et affichage dynamique côté client.
- 💾 Base de données SQL pour stocker les livres, catégories et emprunts.
- 🖼️ UML : Use Case & Diagrammes de classes.

---

## 💡 Architecture

- **Frontend** :  
  - HTML5 / CSS3 / JavaScript (DOM manipulation).
  - Interface utilisateur dynamique avec filtrage, tri et pagination.
  
- **Backend** :  
  - Node.js (sans framework).
  - Gestion des routes, logique métier et connexion à la base de données.

- **Base de données** :  
  - SQL (MySQL / MariaDB).
  - Tables : `livres`, `categories`, `utilisateurs`, `emprunts`.
